# 💰 Finance Dashboard

A modern and responsive Finance Dashboard built with **React.js** and **Vite** to help users visualize and monitor financial data through interactive charts and a clean user interface.

## 🚀 Live Demo

🔗 https://finovate-xi.vercel.app

---

## 📌 Features

- 📊 Interactive financial charts and analytics
- 💵 Income, Expense, and Balance overview
- 📈 Real-time data visualization
- 📱 Fully responsive design
- ⚡ Fast performance with Vite
- 🎨 Clean and modern UI
- 📋 Well-structured React components

---

## 🛠️ Tech Stack

- React.js
- Vite
- JavaScript (ES6+)
- CSS
- Recharts

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/tripathipalak/Finance-Dashboard.git
```

Navigate to the project folder

```bash
cd Finance-Dashboard
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
